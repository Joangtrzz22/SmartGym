# SmartGym

## Descripción

SmartGym es una aplicación móvil desarrollada con Ionic, Angular y Capacitor como proyecto final de la asignatura Programación de Dispositivos Móviles (ISW-307). La aplicación permite gestionar rutinas de entrenamiento, visualizar un mapa del gimnasio, acceder a contenido multimedia y administrar la configuración del perfil del usuario.

---

## Tecnologías utilizadas

- Ionic Framework
- Angular
- Capacitor
- TypeScript
- HTML
- SCSS
- Leaflet
- OpenStreetMap
- @ionic/storage-angular
- @capacitor/geolocation
- @capacitor/network
- @capacitor/camera

---

## Instrucciones de instalación

Clonar el repositorio:

```bash
git clone https://github.com/Joangttrzz22/SmartGym.git
```

Entrar a la carpeta del proyecto:

```bash
cd SmartGym
```

Instalar las dependencias:

```bash
npm install
```

Ejecutar la aplicación:

```bash
ionic serve
```
